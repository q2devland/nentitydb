Create a project

You begin by creating a database and an .NET Core console application, and setting up your environment.

Create database

To create a new database, you can follow the documentation of each database provider:

  • Microsoft SQL Server

  • Oracle Database

  • MySQL

  • MariaDB

  • Percona Server

  • PostgreSQL

  • IBM Db2

  • SQLite

  • Firebird

Creating database schema

Now, you can create the database schema with one of the following scripts. You must replace the string “your-schema” with the current schema of the database.

Set up your environment

Using the Console application template that ships with the .NET Core, generates a stripped down basic application with absolute minimum references required for an application.

Adding a settings file

In order to enable our Console Application to read Configuration from files we will need to add a reference to the following nuget packages:

  • Microsoft.Extensions.Configuration

  • Microsoft.Extensions.Configuration.FileExtensions

  • Microsoft.Extensions.Configuration.Json

Create a simple appsettings.json to store our configuration settings, and add keys and values for the connection string and schema parameters.

applicationsettings.json
{
  "ConnectionStrings": {
    "FirstApp": "your connection string"
  },
  "Schema": "your schema"
}

We’ll slightly ammend our Main method to read our new appsettings.json file.

Program.cs
public class Program
{
    public static void Main(string[] args)
    {
        IConfiguration configuration = new ConfigurationBuilder()
            .SetBasePath(Directory.GetCurrentDirectory())
            .AddJsonFile("appsettings.json")
            .Build();

        Console.ReadKey();        
    }
}

Registering a NEntityDb license

Create a simple license.json to store your NEntityDb license, and copy the license data.

You must install the NEntityDb provider nuget package you want to work with. For more information about install NEntityDb, see Install NEntityDb on .NET.

license.json
{
  "data": {
    "licensedTo": "....",
    "licenseType": "....",
    "id": ".....",
    "providerType": ".....",
    "expirationDate": "...."
  },
  "signature": ".................."
}

Register the NEntityDb license in the Main method.

Program.cs
public class Program
{
    public static void Main(string[] args)
    {
        IConfiguration configuration = new ConfigurationBuilder()
            .SetBasePath(Directory.GetCurrentDirectory())
            .AddJsonFile("appsettings.json")
            .Build();        

        DbLicense.LoadFromFile(Path.Combine(Directory.GetCurrentDirectory(), "license.json"));

        Console.ReadKey();
    }
}

For more information about register a NEntityDb license, see Register a license.
