Licensing

The license is a plain text JSON file that contains details such as the product name, subscription expiry date and so on. The file is digitally signed, so don't modify the file. Even inadvertent addition of an extra line break into the file will invalidate it. You need to set a license before utilizing NEntityDb. It is only required to set a license once per application (or process). The license can be loaded from a file, or string.

For more information about register a license, see Register a license.

Trial license

To evaluate NEntityDb you don’t need our permission – all features are unlocked. You just need obtain a trial license from here. Once you’re ready to begin development, please purchase an appropriate license key.

Keep the following points in mind when using your trial license:

  • With a single trial license you can evaluate all database providers.
  • Trial license has a 30 day duration.
  • You must register trial license as a standard license.

Frequently Asked Questions

What's included in the license?

NEntityDb product licenses are perpetual and within the first 12 months include product updates (new features and fixes) and technical support.

Do I have to renew my subscription/license after a year?

Renewing your NEntityDb product is optional. You can continue to use the version you own forever, however, renewing will ensure you get additional updates in the coming year and extends your product support for an additional year.

Now many licenses do I need?

Each license allows 3 developers to work with the product.

Can I use the product on more than one project/server/domain?

NEntityDb product licenses have no deployment limitations - they can be used for an unlimited number of applications spanning various servers and domains.
