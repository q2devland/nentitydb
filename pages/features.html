Features

Cross-plaform

NEntityDb is a lightweight, extensible, and cross-platform data access technology built on the .NET platform.

.NET is a free, cross-platform, open source developer platform for building many different types of applications. With .NET you can:

  • Create applications for Windows, macOS, and Linux
  • Use multiple languages, editors, and libraries to build your application
  • Build application for web, mobile, desktop, gaming, and IoT
View more

Database abstraction layer

NEntityDb provides you with a database abstraction layer (an application programming interface) which:

  • Unifies the communication between a computer application and databases
  • Reduces the amount of work by providing a consistent API to the developer
  • Hides the database specifics behind this interface as much as possible
  • Allows you to change the database (RDMS) of your application without having to rewrite most of your database queries
View more

Productivity

There are a number of productivity benefits to using NEntityDb for development of databased applications:

  • Code reduction
  • Application design
  • Application maintainability
  • Portability
View more

.NET LINQ Expressions

NEntityDb uses .NET LINQ Expressions to query and save data from database. The .NET LINQ Expressions enables language-level code expressions to be represented as objects in the form of expression trees.

.NET LINQ Expressions allows:

  • Write strongly typed queries based on your entities (code that is easy to understand and maintain)
  • Catching errors at compile time (prevents type mismatch errors being caught when the code is compiled rather at run-time)
View more

Fluent API

With NEntityDb you can use Fluent APIs both to configure your model and to query and save data. Using Fluent APIs in NEntityDb makes code readability easier.

Three reasons why we use Fluent APIs in NEntityDb:

  • Readability is a primary design concern
  • Fluent APIs emphasize the purpose of your code
  • Fluent APIs let you wield complex concepts with ease
View more

Static and dynamic queries

NEntityDb allows you to query or save data both statically and dynamically.

  • Static queries with .NET LINQ Expressions prevents type mismatch errors
  • Any statically defined query can be defined dynamically, and vice versa
  • You can also add dynamically parts to the definition of a static query
